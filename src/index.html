Featured: AI IT Support Chatbot

This app is a GPT-based IT support system that uses context and previous examples to solve IT problems for business users.

It utilizes OpenAI's GPT-3.5-turbo and a Postgres database to store users, supported apps, and troubleshooting examples.

The system learns from from positive reinforcement by summarizing the problem it helps solve, then adding that to the troubleshooting examples.

This is my first full-stack project, Django project, and GPT-based project, and I've learned a lot making it!
